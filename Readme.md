# rcrypto

A rust cryptography library

## Support Algorithms

- [x] MD5;
- [x] SHA;
- [x] SM3;
- [x] AES;
- [x] DES;
- [x] Keccak;
- [x] SHA3;
- [x] HMAC;
- [x] BlockCipherMode;
  - [x] ECB/CBC/CFB/OFB/CTR;
- [x] TDES;
- [x] CMAC;
- [x] ZUC/ZUCCipher/ZUCMac;
- [x] SM4;
- [x] RSA;
- [x] EllipticCurve;
- [x] DSA;
